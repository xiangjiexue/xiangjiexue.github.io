Something about R

R is a language and environment for statistical computing and graphics. It is a GNU project which is similar to the S language and environment which was developed at Bell Laboratories (formerly AT&T, now Lucent Technologies) by John Chambers and colleagues.

R provides a wide variety of statistical (linear and nonlinear modelling, classical statistical tests, time-series analysis, classification, clustering, …) and graphical techniques, and is highly extensible.

Packages

npfixedcompR

Note that this package will be freezed. The npfixedcomp2 is under ongoing development.

During my PhD study, I have developed different approaches to estimating a mixing distribution (or mixtures) non-parametrically and implemented these in R. The source code can be found on this Github repo and can be installed using the devtools package.

devtools::install_github("xiangjiexue/npfixedcompR")

non-parametric mixtures

According to Lindsay (1983), The non-parametric mixture model assumes that the distribution of all underlying incidental parameters is treated as completely unspecified as to whether it is discrete, continuous or in any particular family of distributions.

We use the \(z\)-values of the prostate data mentioned in Singh et al. (2002) as an example. There are 6033 observations and we are interested in investigating the proportion of null effects (\(z\)-statistic is 0) and the non-null effects.

\(z\)-values/\(z\)-statistics

The two-sample t-test with pooled variance is used in Singh et al. (2002), and \(t\)-statistics are obtained. One can use the cumulative distribution function (CDF) to compute the probability for rejecting the null hypothesis, but as mentioned in Efron (2010), a more conventional way is to transform to \(z\)-statistics.

null hypothesis/null effect

A null hypothesis is a type of hypothesis used in statistics that proposes that there is no difference between certain characteristics of a population (or data-generating process). In the case of \(t\)-test and \(z\)-test, the null effect corresponds to \(t\)-statistics and \(z\)-statistics being 0.

With no assumption of the underlying structure of the null effects, non-parametric method should be used. With this R package, we can estimate the distribution of the non-null effect given the null proportion. In this case, we assume the component density is normal and use non-parametric maximum likelihood to find the mixing distribution.

npfixedcomp2

This is package is to replace npfixedcompR in the future. The npfixedcompR package is written in R, it is relatively slow and has some issue. This package is written in C++ using the Eigen library.

The homepage of this package can be found here and the source code of this package can be found here.

For now, nptll, npnormll, npnormllw, npnormcvm, npnormcvmw, npnormad and npnormcll have been implemented and it is uniformly faster than the corresponding function in npfixedcompR.

detach("package:npfixedcompR", unload=TRUE)
library(npfixedcomp2)
